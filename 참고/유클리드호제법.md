> 💡 유클리드 호제법/알고리즘(Euclidean Algorithm)

- 두 수의 '최대공약수(GCD)'를 찾기 위한 알고리즘을 의미합니다.

- 큰 수를 작은 수로 나누어 떨어지게 한 뒤, 수를 반복적으로 수행하여 나머지 0이 될때까지 작동하는 방법을 의미합니다.이때 작은 수가 최대공약수입니다.

```javascript

/** 최대공약수 */
const gcd = (a, b) => {
  if(b === 0) {
    return a;
  }
  return gcd(b, a % b);
}

//* 최대공배수 */
const lcm = (a, b) => {
  return (a * b) / gcd(a, b);
}
```
